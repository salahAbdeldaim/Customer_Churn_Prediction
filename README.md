# Customer Churn Prediction

## Project Overview

This project predicts customer churn using Machine Learning techniques.

## Dataset

The dataset contains customer information such as:

* Age
* Support Calls
* Payment Delay
* Subscription Type
* Contract Length
* Total Spend

## Workflow

1. Data Cleaning
2. Missing Value Handling
3. One-Hot Encoding
4. Feature Selection
5. Feature Engineering
6. Model Training
7. Model Comparison
8. Hyperparameter Tuning
9. Final Evaluation

## Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* Extra Trees
* XGBoost
* CatBoost

## Best Model

Random Forest Classifier

## Final Accuracy

93.45%
